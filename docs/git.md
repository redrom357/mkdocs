## Git

### Git Quick Start Commands

#### Check Git version

```bash
git version
```

#### Create a local Git repo

```bash
git init
```

#### Add a file to Git

```bash
git add file
```

#### Add all file to Git

```bash
git add *
```

#### Add all file to Git

```bash
git add *
```

#### Commit a file or files

```bash
git commit -m "Adding initial file(s)"
```

#### Ignoring file(s) in Git

```bash
vim .gitignore
git add .gitignore
git commit -m "adding ignore file"
```

#### Check git log

```bash
git log
# or
git log --oneline
```

#### Check git status

```bash
 git status
```
